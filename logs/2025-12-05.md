### 🔹第17章：システム企画
- **RFI / RFP**
  - RFI：情報提供依頼 → 事前調査用
  - RFP：提案依頼 → ベンダ選定用
- **調達系の用語整理**
  - CSR調達：企業の社会的責任を考慮
  - グリーン調達：環境配慮した材料・製品の調達
  - グリーン購入：消費者視点で環境負荷低い製品を選択
- **要件整理**
  - 機能要件 / 非機能要件（性能・可用性・保守性など）

### 🔹第18章：経営戦略マネジメント
- **SWOT分析**
  - 内部（Strength/Weakness）、外部（Opportunity/Threat）
- **PPM分析（プロダクトポートフォリオマネジメント）**
  - 市場成長率 × 市場占有率で分類  
  - 花形 / 金のなる木 / 問題児 / 負け犬  
  - 「規模の経済」の概念あり
- **PLC（プロダクトライフサイクル）**
  - 導入 → 成長 → 成熟 → 衰退
- **コトラーの競争戦略**
  - リーダ / チャレンジャ / フォロワ / ニッチャ
- **バランススコアカード（BSC）**
  - 財務 / 顧客 / 業務プロセス / 学習と成長
- **マーケティング**
  - 4P：Product / Price / Place / Promotion
  - 4C：Customer Value / Cost / Convenience / Communication  
    → 4Pと対応関係で整理
- **価格決定方式**
  - コストプラス価格決定法  
    → 販売価格 = 製造原価 + 利益
- **経営管理システム**
  - ERP（基幹業務統合）
  - SCM（供給管理）
  - CRM（顧客管理）
- **ナレッジマネジメント**
  - 暗黙知 → 形式知 → 共有化

### 🔹第19章：ビジネスインダストリー
- **スマートグリッド / HEMS**
  → エネルギーマネジメント系
- **生産方式**
  - JIT（ジャストインタイム：必要な時に必要量だけ）
  - セル生産方式（少量多品種 / 柔軟性）
- **MRP（資材所要量計画）**：生産計画から必要資材を算出
- **eビジネスと用語**
  - 逆オークション：買い手が価格提示し、売り手が落札
  - G2B（行政 → 企業）
  - O2O（オンライン → 実店舗）
  - エスクローサービス（仲介）
  - RFID：梱包の上からも読み取り可能
  - ロングテール：売れ行き少数商品 × 種類大量 = 売上大
  - EDI：企業間データ交換
  - CGM：消費者生成メディア
  - シェアリングエコノミー：資産の共同利用
- **IoTシステム**
  → センサ、ネットワーク、クラウド連携の理解
